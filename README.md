## Getting Started
### 1. Install PNPM
```
npm install -g pnpm 
```

### 2. Clone Repository
```
git clone https://github.com/username/myportfolio.git
cd myportfolio
```

### 3. Install dependencies
```
pnpm install
```

### 4. Start development server
```
pnpm run dev
```
Visit http://localhost:5173 to see the website.
